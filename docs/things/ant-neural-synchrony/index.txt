10:T4e9,<p>Ants use visual information to quickly and accurately learn routes through their environment, despite a small brain and a low-resolution visual system. This navigation may be driven by a search for familiarity between a current view and the views previously experienced along the target route. There is little consensus on how this familiarity measure is implemented, but a recently proposed general familiarity measure, whereby an input history is encoded in a spiking neural network and the synchrony of spike timing is measured for a new input, seems promising. This project evaluates the use of this measure by extending it to a basic navigation task, using real images and various metrics. Its performance is found to be relatively weak, but experimental shortcomings and the plausibility of the method show that further investigation is warranted.</p>
